# AI Agents 設定

## 基本設定
- AI エージェントによる回答は、必ず日本語でお願いします。

## ワークフロールール

### タスク管理
- 複雑なタスクは段階的に分解して実行する
- 各ステップの進捗を明確に報告する
- 不明点があれば、推測せずにユーザーに確認する

### 効率化ルール
- 並列実行可能な処理は並列で実行する
- 同じ情報を複数回取得しない
- キャッシュ可能な情報は再利用する

### コード変更の原則
- 既存のコーディング規約に従う
- 変更前に影響範囲を確認する
- セキュリティリスクを考慮する
- テストがある場合は必ず実行する

### エラー対応
- エラー発生時は詳細を報告する
- 同じエラーを繰り返さない
- 解決できない場合は代替案を提示する

## コミュニケーション
- 簡潔で分かりやすい説明を心がける
- 技術的な詳細が必要な場合のみ、詳細な説明を行う
- ユーザーの意図を正確に理解することを優先する
